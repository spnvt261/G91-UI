# 2. GLOBAL API RESPONSE FORMAT (NEW)

All backend APIs now return a unified envelope:

{
"code": "SUCCESS",
"message": "...",
"data": {},
"errors": []
}

Rules:

SUCCESS response:

code = "SUCCESS"
data contains payload

Error response:

code != SUCCESS
errors may contain field validation messages

You must update axios interceptors and service parsing to match this format.
